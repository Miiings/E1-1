
## 3. 파일 생성 및 조작

```bash
touch test.txt
cp test.txt copy.txt
mv copy.txt moved.txt
rm moved.txt
touch a.txt
touch b.txt
cp a.txt ./b.txt
cp a.txt ../../Desktop
```
